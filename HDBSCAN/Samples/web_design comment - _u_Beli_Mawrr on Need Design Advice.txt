Yes! This is 100% a php/mysql with POST operations. You'll probably need a little bit of experience with html, JQuery, php, and SQL. This is not actually a design question but a system/programming question You'll need to set up a javascript thing (look up set interval) that will constantly run a jQuery POST command to a php script (again look that up), which will get the list of checked boxes from the server. Then, a separate php script will be executed whenever a user checks a box, which transmits the name of the checker to the server and updates the list of checked items. Each of these can be broken down in detail or with code if you like.
