First of all, WPF, right? You want to create DataTemplates and a DataTemplateSelector Typically, you create a DataTemplateSelector when you have more than one DataTemplate for the same type of objects and you want to supply your own logic to choose a DataTemplate to apply based on the properties of each data object.  https://msdn.microsoft.com/en-us/library/system.windows.controls.datatemplateselector(v=vs.110).aspx ListView example: http://dotnetpattern.com/wpf-listview-itemtemplateselector Create however many DataTemplates you feel you need and in your DataTemplateSelector assign a DataTemplate based on type.
